# ruby-sparkpost

SparkPost client library for Ruby https://www.sparkpost.com

## Installation

Add this line to your application's Gemfile:

    gem 'ruby-sparkpost'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruby-sparkpost
